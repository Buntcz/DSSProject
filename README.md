## Prerequisites to run the project
- Node v24.14.1
- npm version 11.11.0
If you have these two you are ready to download the project

## To download the project click download as ZIP above the repo files like in the image
![Download](download.png?raw=true "Download")


## After you've downloaded the project confirm you have these files
- package.json

## Download all the dependencies using npm install

Before downloading the dependencies make sure you are in the dss-project folder. If not open a cmd in it or if on linux run:

```console
cd dss-project
```

Then run this command in cmd/terminal

```console
npm install
```

This will install all the dependencies needed to run the project

## Running the project
To run the project run this command in the cmd/terminal

```console
npm run dev
```

## Accessing the project 
After you've ran the previous command a link will show up in the cmd terminal which you need to open in the browser.
It should look like this.

![Run the project](run.png?raw=true "Run the project")

Either click the link or open it in your browser.

To stop the server from running just click CTRL + C while accessing the terminal/cmd.

# And you are finished. Enjoy the use of the todo app.
